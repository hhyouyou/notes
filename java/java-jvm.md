### 各个模块简介

#### 虚拟机栈

**线程私有**，生命周期和线程相同。每条线程都有一个独立的栈（VM Stack）。每个方法执行时都会创建一个栈帧，也就是说，当有一条线程执行多个方法的时候，就有一个栈，栈中有多个栈帧。

#### 本地方法栈

#### 程序计数器

**线程私有**

#### 堆Heap

**线程共享**， 几乎所有对象实例以及数组都要在堆上分配。
