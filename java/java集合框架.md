# 概述

本章准备记录java的集合框架，

Collection：存储对象的集合

Map：存储键值对映射表



## Collection



### 1. Set

* TreeSet : 基于红黑树实现，支持有序操作。查找的效率是 O(logN)
* HashSet：基于哈希表实现，支持快速查找，无序。查找效率是O(1)
* LinkedHashSet：具有HashSet的查找效率，内部使用双向链表维护元素的插入顺序。
